Tools -
- CARTA (https://cartavis.github.io/) - Data Visualization tool, can be used for exploring the fits files
- SoFiA 2 (https://github.com/SoFiA-Admin/SoFiA-2/wiki) - Command line tool to find the HI sources given a fits file. Look into the Manual (https://raw.githubusercontent.com/wiki/SoFiA-Admin/SoFiA-2/documents/SoFiA-2_User_Manual.pdf) and paper (https://arxiv.org/abs/1501.03906)
- Anaconda (https://www.anaconda.com/products/individual) - Python based data science toolkit.
- Python 3 (https://www.python.org/downloads/)
- Astropy (https://www.astropy.org/) - Python library for astronomy. Will be useful for handling fits files and some basic machine learning analysis.
